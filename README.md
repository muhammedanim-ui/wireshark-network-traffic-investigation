Traffic Investigation With Wireshark

## Introduction

In this project, an investigation of network traffic is carried out using Wireshark on a Kali Linux virtual machine. The traffic is captured and analyzed to learn about how different network protocols work and how external IPs can be analyzed.

## Tools Used

- Kali Linux
- Oracle VirtualBox
- Wireshark
- AbuseIPDB

## Skills Shown

- Packet Capturing
- Network Traffic Analysis
- DNS Analysis
- TCP and TLS Analysis
- External IP Analysis
- Technical Reporting
